Haml 是什麼？

by 查斯特

Haml（HTML Abstraction Markup Language)HTML 抽象標記式語言是一種模板系統，它可以避免在 Web 文件中內嵌代碼，使 HTML 更簡易和乾淨 Haml 為 HTML 中提供了一些動態內容的靈活性。為執行 Haml 代碼，檔案需要採用 .haml 副檔名。這些檔案類似 .erb 檔案，這有助於在開發網路應用程式時嵌入 Ruby 代碼。在解析代碼注釋時，Haml 採用與 Ruby 1.9 或之後版本相同的規則 Haml 只能理解相容 ASCII 的編碼（例如UTF-8），而不能理解不相容ASCII的UTF-16與UTF-32 來個 Hello-World 例子HTML: 1<p class="sample" id="welcome" > Hello, World!!! </p> Haml: 1%p{:class => "sample", :id => "welcome"} Hello, World!!! 又可以變成：
